
# Example Markdown

## Headings
Markdown supports multiple heading levels from `#` to `######`.

## Text Formatting
- **Bold text** with `**text**`
- *Italic text* with `*text*`
- ~~Strikethrough~~ with `~~text~~`
- `Inline code` with backticks

## Lists

### Unordered List
- Item 1
- Item 2
- Item 3

### Ordered List
1. First item
2. Second item
3. Third item

## Code Block
```javascript
function hello(name) {
    console.log(`Hello, ${name}!`);
}
```

## Links and Images
[GitHub](https://github.com)

> This is a blockquote example.
